# VibeMap Backend

FastAPI backend scaffold for the chatbot and community APIs.

## Chatbot Structure
- `app/api/chat.py`: chat router
- `app/schemas/chat.py`: request/response schemas
- `app/services/chat_service.py`: business logic
- `app/services/openai_service.py`: OpenAI integration
- `app/main.py`: FastAPI app entrypoint
